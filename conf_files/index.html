Welcome to Your Monerodo!

 

terminal - access the terminal, where you will find the Monerodo OS text user interface. Default user is bob, and the password is vB3R9mDC. You should really change the password on the first use.

 

pool - access your pool stats page, if you have it running

 

wallet - access your MiniNodo wallet, if you have it running

 

admin - access the Webmin admin panel for your Monerodo (not very useful, terminal is better).

 

ajenti - access the Ajenti admin panel for your Monerodo (this can be buggy. It might not work)

 

File an issue / bug report / feature request - I won't know it doesn't work if you don't tell me.
